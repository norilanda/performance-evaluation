# Task 1: Play With Timers

The purpose of this lab assignment is to provide basic experience with simple experiments.
As a case study, we will investigate various properties of multiple timing sources.

- [The notebooks task](00-notebooks) provides introductory information on how to use
  R notebooks in the lab (these are not mandatory but recommended).

- [The timers task](01-timers) is about investigating properties of time query functions.

- [The sleeping task](02-sleeping) is about investigating properties of sleep functions.
